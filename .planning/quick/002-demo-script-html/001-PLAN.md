# Plan: Judge-Friendly Demo Script HTML

## Goal
Create presenter/demo-script.html -- a step-by-step presentation guide for hackathon judges showing the full BoothApp pipeline with talking points and arrow key navigation.

## Success Criteria
1. HTML page at presenter/demo-script.html
2. 5 demo steps: (1) Badge scan, (2) Chrome extension clicks, (3) Real-time session progress, (4) AI analysis, (5) Generated follow-up email
3. Talking points for each step
4. Arrow key navigation between steps
5. Dark theme matching existing presenter pages
6. Works standalone (no external dependencies beyond existing components)
