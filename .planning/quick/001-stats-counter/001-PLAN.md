# Stats Counter in Presenter Header

## Goal
Add a stats counter strip in the presenter landing page header showing four KPIs:
Total Sessions, Average Score, Products Demonstrated, Follow-Up Actions Generated.
Numbers count up from 0 with easing animation. Auto-update as mock data refreshes.

## Success Criteria
1. Four stat counters visible in header area below tagline/live badge
2. Count-up animation from 0 with easing (ease-out cubic, matching existing pattern)
3. Auto-updates every 5s refresh cycle alongside existing data
4. Visual design consistent with existing dark theme and red/blue/green accents
5. Responsive layout adapts to mobile
