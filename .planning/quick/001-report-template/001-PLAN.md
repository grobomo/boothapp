# Plan: Presentation-Quality HTML Report Template

## Goal
Create a polished, presentation-quality HTML report template for booth visitor analysis with Trend Micro Vision One branding, card-based layout, confidence badges, product timeline, and print-friendly styling.

## Success Criteria
1. Dark header with Trend Micro Vision One branding (logo, colors)
2. Card-based layout for visitor info, products shown, interests, recommendations
3. Color-coded confidence badges: high=green, medium=yellow, low=red
4. Products Demonstrated timeline with click sequence and timestamps
5. Recommended Follow-Up Actions section with checkboxes
6. Print-friendly CSS for PDF export (@media print)
7. Professional typography with dark header, light body
8. Python module at analysis/engines/report_template.py that generates complete HTML
9. Demo script proves the template renders correctly
