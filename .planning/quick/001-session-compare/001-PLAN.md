# Session Comparison View

## Goal
Add a side-by-side session comparison view at `presenter/compare.html` with `presenter/lib/compare.js` that lets judges compare 2-3 booth sessions.

## Success Criteria
1. `presenter/compare.html` exists and loads without external dependencies
2. `presenter/lib/compare.js` handles data loading and rendering
3. Users can select 2-3 session folders to compare
4. Shows: visitor names, engagement scores, product counts, key interests, top recommendations
5. Responsive grid layout
6. Dark theme consistent with existing demo.html styling
7. Data sourced from `output/summary.json` in each session folder
