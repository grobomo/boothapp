# Session Handoff Page

## Goal
Create presenter/handoff.html -- a clean handoff document for SDR/account teams after a booth session is analyzed. Includes visitor profile, buying signals, follow-up timeline (day 1/3/7/14), and pre-written email drafts for each touchpoint.

## Success Criteria
1. handoff.html loads sample_data.json and renders a complete handoff document
2. Visitor profile section with all visitor fields
3. Buying signals section derived from interests (with confidence levels)
4. 4-phase follow-up timeline: Day 1, Day 3, Day 7, Day 14 with specific actions
5. Pre-written email drafts for each touchpoint (copy-to-clipboard)
6. Visual style matches existing V1 branding (dark header, red accents, clean cards)
7. Print-friendly layout
8. Works standalone in browser with no build step
