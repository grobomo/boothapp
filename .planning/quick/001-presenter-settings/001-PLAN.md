# Presenter Settings Page

## Goal
Create a settings page at `presenter/settings.html` that allows configuring S3, Lambda, model, screenshot quality, auto-refresh, theme, notifications, and admin password. Store in localStorage, load defaults from config.json. Include Test Connection button.

## Success Criteria
1. Settings page exists at presenter/settings.html
2. Matches existing presenter page styling (dark theme, red accents, bg-grid)
3. All 8 config sections present and functional
4. Settings persist in localStorage
5. Defaults loaded from presenter/config.json
6. Test Connection button verifies S3 + Lambda access
7. Theme toggle switches dark/light
8. Admin password change with current/new/confirm fields
