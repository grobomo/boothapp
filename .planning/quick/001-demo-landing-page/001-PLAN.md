# Demo Landing Page

## Goal
Create a visually stunning demo landing page at `presenter/demo.html` for trade show presentations. Displayed on a big screen during live demos, showing real-time session activity with Trend Micro branding.

## Success Criteria
1. Large BoothApp logo and tagline "AI-Powered Trade Show Demo Capture"
2. Real-time status cards: Active Sessions, Total Demos, Reports Generated
3. Live activity feed with recent events (session starts, analysis, emails)
4. Dark theme with Trend Micro brand colors (red #D71920, white)
5. Auto-refresh every 5 seconds
6. Fetches from Lambda endpoint, falls back to mock data
7. Visually impressive: animations, particle effects, professional typography
