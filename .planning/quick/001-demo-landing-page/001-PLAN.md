# Demo Landing Page

## Goal
Create a visually stunning demo landing page at `presenter/demo.html` for trade show presentations. The page displays on the big screen during live demos, showing real-time session activity with Trend Micro branding.

## Success Criteria
1. Large BoothApp logo and tagline "AI-Powered Trade Show Demo Capture" prominently displayed
2. Real-time status cards showing: Active Sessions count, Total Demos count, Reports Generated count
3. Live activity feed showing recent events (session starts, analysis completions, email generations)
4. Dark theme with Trend Micro brand colors (red #D71920, white)
5. Auto-refresh every 5 seconds
6. Fetches data from session orchestrator Lambda endpoint, falls back to mock data if unavailable
7. Visually impressive for a trade show audience (animations, professional typography, polished layout)
