# Popup UI Redesign

## Goal
Redesign the Chrome extension popup with professional dark theme matching the BoothApp presenter UI. Show session status, visitor info, counters, and S3 connection status using Trend Micro brand colors.

## Success Criteria
1. Clean header with BoothApp logo (SVG) and version number
2. Session status indicator: green dot = recording, red dot = stopped
3. Visitor name display when session is active
4. Click counter badge with count
5. Screenshot counter badge with count
6. S3 connection status indicator
7. Dark background (#0B0E14) with Trend Micro red (#D71920) accents
8. Manifest V3 Chrome extension with proper popup dimensions
