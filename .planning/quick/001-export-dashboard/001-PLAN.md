# Export Dashboard

## Goal
Create a data export dashboard at `presenter/export.html` for exporting session data.

## Success Criteria
1. CSV export (one row per session with key metrics)
2. JSON export (full data)
3. PDF report (browser print with styled print CSS)
4. Zip download per session (all files packaged)
5. Bulk email draft export
6. Table with checkboxes for selection
7. Export selected or export all
8. Download progress indicator
