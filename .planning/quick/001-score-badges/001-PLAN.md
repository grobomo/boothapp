# Score Badges - Visitor Engagement Scoring

## Goal
Add visitor engagement scoring with visual indicators (colored pill badges) throughout the BoothApp UI, plus a Score Distribution chart on the analytics section.

## Success Criteria
1. `presenter/components/score-badge.js` exists with reusable ScoreBadge component
2. Color coding: green (80-100), yellow (60-79), red (0-59)
3. Score fill animates on page load
4. Tooltip shows score breakdown (engagement, coverage, follow-up)
5. Score badges appear on: session list cards, session viewer header, analytics cards
6. Score Distribution chart shows histogram across all sessions
7. All existing functionality unbroken
