# Popup UI V2

## Goal
Improve Chrome extension popup UI with session management, counters, and S3 status.

## Success Criteria
1. Session status indicator: green=active, gray=idle, red=error with session ID display
2. Click counter showing tracked clicks this session
3. Screenshot counter showing captured screenshots
4. Start/Stop session button that signals background.js
5. Settings gear icon toggling S3 configuration panel
6. S3 connection status indicator (green=configured, red=not configured)
7. Dark theme matching presenter dashboard aesthetic
8. State persisted via chrome.storage
