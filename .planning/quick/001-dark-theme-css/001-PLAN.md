# 001 - Dark Theme CSS Framework

## Goal
Create a unified dark-themed CSS design system at `presenter/styles/theme.css` with custom properties, glassmorphism cards, buttons, tables, forms, grid utilities, typography, and animations. Import it in all presenter HTML pages.

## Success Criteria
1. `presenter/styles/theme.css` exists with all required components
2. CSS custom properties for colors (--bg-primary, --bg-secondary, --accent, --text, --success, --warning, --error)
3. Glassmorphism `.glass-card` class with backdrop-filter, border, shadow
4. Button styles: `.btn-primary`, `.btn-secondary`, `.btn-danger`
5. `.data-table` with alternating row colors
6. Form input styles
7. Grid layout utilities
8. Typography scale
9. Animation keyframes: fadeIn, slideUp, pulse
10. All presenter HTML pages import theme.css
