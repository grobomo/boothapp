# Popup UI Upgrade

## Goal
Redesign Chrome extension popup (popup.html + popup.js) with Trend Micro branding, session status indicators, counters, S3 status, and settings toggle.

## Success Criteria
1. Trend Micro branding with red accent colors (#D12B2B) and dark sidebar feel
2. Session status indicator: Idle (grey), Recording (red pulsing dot), Uploading (blue spinner), Complete (green check)
3. Click counter and screenshot counter for current session
4. S3 connection status with green/red dot
5. Settings gear icon toggles S3 config section visibility
6. Start Demo / End Demo buttons only appear when S3 is configured
7. Clean CSS: border-radius, box-shadows, smooth transitions
8. Extension directory created in boothapp repo with all files from boothapp-clone
