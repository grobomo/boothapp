# Visitor Summary Card Component

## Goal
Create a standalone visitor summary card HTML component at `presenter/components/visitor-card.html` that displays visitor details, session metrics, products demonstrated, and an AI-generated summary in a dark-themed modern card UI.

## Success Criteria
1. File exists at `presenter/components/visitor-card.html`
2. Displays: visitor name (large), company, badge photo thumbnail
3. Shows: session duration, click count
4. Lists products demonstrated (derived from clicks data)
5. Includes 3-sentence AI-generated summary placeholder
6. Modern card UI: box-shadow, border-radius, dark theme
7. Consistent with existing BoothApp design system (CSS vars from popup.html/demo.html)
8. Self-contained HTML file that can be embedded or opened standalone
