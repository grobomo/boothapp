# Visitor Check-in Kiosk Page

## Goal
Create a tablet-optimized visitor self-check-in kiosk page at `presenter/kiosk.html` for trade show booths.

## Success Criteria
1. Large "Welcome to BoothApp" header
2. Camera capture button using getUserMedia for badge photos
3. Manual name/company/title entry fields as fallback
4. "Start Demo" button that POSTs to /api/session to create session
5. QR code display linking to session results page
6. Timer showing estimated wait time
7. Touch-friendly large buttons, works on iPad/Android tablet in landscape
8. Consistent with existing demo.html branding (dark theme, red accents)
