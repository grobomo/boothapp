# Presentation Countdown Timer

## Goal
Add a presentation countdown timer at `presenter/countdown.html` to help the team manage hackathon presentation time.

## Success Criteria
1. Large countdown clock showing MM:SS format
2. Configurable total time via URL param `?minutes=5` (default 5)
3. Color changes: green >2min, yellow 1-2min, red <1min
4. Flash/pulse animation in last 30 seconds
5. Optional sound alert at 1min and 30sec marks
6. Full-screen mode
7. Pause/resume button
8. Show/hide presenter notes below timer
9. Dark theme
