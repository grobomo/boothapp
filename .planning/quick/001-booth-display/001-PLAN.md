# Booth Display Mode

## Goal
Create presenter/booth-display.html -- a full-screen, dark-themed display page optimized for a 55-inch TV at the booth. Shows real-time session stats, recent visitors, and an activity feed with animated particle background.

## Success Criteria
1. Full-screen dark theme, no scrollbars, optimized for 1920x1080 (55" TV)
2. Animated BoothApp title with particle background canvas
3. Live session counter with green pulse animation for active sessions
4. Recent visitor cards showing photo, name, company with fade in/out transitions
5. Scrolling activity feed showing latest events (session started, analysis complete, etc.)
6. Team logo and hackathon branding at bottom
7. Auto-refresh via polling (same API pattern as live.html)
8. No auth required (display-only, read-only)
9. Single self-contained HTML file (consistent with other presenter pages)
