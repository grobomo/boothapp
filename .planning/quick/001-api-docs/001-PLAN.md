# API Documentation Page

## Goal
Add a comprehensive, interactive API documentation page at `presenter/api-docs.html` that documents all BoothApp API endpoints with live "Try it" buttons.

## Success Criteria
1. Page at `presenter/api-docs.html` documents all 7 API endpoints
2. Each endpoint shows: method, URL, request body, response format, example curl
3. Interactive "Try it" buttons make live API calls and display responses
4. Dark theme matching existing presenter pages (#0d1117 / #161b22 palette)
5. JSON syntax highlighting in response display
6. Nav component included, API Docs link added to nav
7. Server starts without errors after changes
