# Demo Landing Page

## Goal
Create a visually striking booth kiosk landing page at `demo/landing/index.html` with Trend Micro branding that attendees see when approaching the booth.

## Success Criteria
1. Full-screen dark theme with Trend Micro branding (red accent, TM logo)
2. Large animated "Welcome to Trend Micro" heading
3. Session counter showing total demos completed today
4. QR code placeholder for visitor to scan
5. Subtle CSS animations (gradient background, pulsing elements)
6. Session-in-progress overlay with spinner when session starts
7. Opens in browser with zero JS errors
