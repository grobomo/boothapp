# Visitor Sentiment Timeline

## Goal
Add a color-coded sentiment timeline bar to the HTML session report that tracks
emotional indicators (positive, neutral, hesitation, skepticism) throughout the
booth conversation transcript.

## Success Criteria
1. Sentiment analysis engine detects 4 sentiment types from transcript segments
2. Color-coded timeline bar renders in the HTML report between products and interests
3. Sample data includes sentiment_timeline field; sample report renders correctly
4. Unit tests cover the analysis engine and the HTML rendering
5. All existing tests still pass
