# Popup Redesign Plan

## Goal
Improve the Chrome extension popup UI with professional Trend Micro branding, proper status states, counters, and conditional UI elements.

## Success Criteria
1. Dark sidebar feel with Trend Micro red accent (#D32F2F)
2. Session status: Idle (grey), Recording (red pulse), Uploading (blue spinner), Complete (green check)
3. Click counter and screenshot counter visible
4. S3 connection status indicator (green=configured, red=not)
5. Gear icon toggles S3 config section
6. Start Demo / End Demo button only visible when S3 configured
7. Clean CSS: border-radius, box-shadows, smooth transitions
8. No console errors when loaded in Chrome
