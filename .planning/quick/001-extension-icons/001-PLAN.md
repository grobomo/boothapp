# Extension Icons & Branding

## Goal
Create Chrome extension icons (16, 48, 128px) with BoothApp branding and update the popup header.

## Success Criteria
1. extension/icons/icon-16.png, icon-48.png, icon-128.png exist and are valid PNGs
2. Icons show a stylized "B" in Trend Micro red (#D71920) on dark background
3. manifest.json references all three icon sizes
4. scripts/generate-icons.sh produces the icons reproducibly
5. Popup header shows the icon and "BoothApp" branding
