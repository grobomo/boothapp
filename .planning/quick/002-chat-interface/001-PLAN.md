# Chat Interface for BoothApp

## Goal
Build a chat interface that lets SEs quickly query session data, get summaries, and manage sessions through conversational commands -- reducing the time to find and act on information.

## Success Criteria
1. Chat page at `/chat.html` with dark theme matching existing UI
2. Integrated into nav bar
3. Server-side chat API with intent detection and session queries
4. Quick command shortcuts for common operations
5. Responsive design (mobile + desktop)
6. Chat history support
