# Session Share Page

## Goal
Create `presenter/share.html` - a public-safe, shareable view of a demo session that can be sent to visitors after demos. Clean white/light theme, professional layout.

## Success Criteria
- [ ] `share.html?session=<id>` loads session data from S3
- [ ] Shows visitor greeting with their name
- [ ] Shows products demonstrated with brief descriptions
- [ ] Shows key discussion points from the session
- [ ] Shows recommended next steps
- [ ] Shows SE contact card
- [ ] Has "Schedule a follow-up" mailto button
- [ ] Does NOT show raw transcript or internal scores
- [ ] Clean, professional white/light theme
- [ ] Generates unique shareable URL per session
- [ ] Server route serves the page without auth
- [ ] Mobile responsive
