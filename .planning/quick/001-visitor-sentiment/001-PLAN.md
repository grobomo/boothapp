# Visitor Sentiment Analysis

## Goal
Add a visitor sentiment analysis engine that analyzes visitor emotional state throughout a demo interaction and writes structured output to `output/sentiment.json`.

## Success Criteria
1. `analysis/engines/prompts.py` exists with sentiment analysis logic
2. Analyzes: initial engagement level, peak interest moments, hesitation/skepticism signals, overall buying temperature (cold/warm/hot)
3. Writes structured JSON output to `output/sentiment.json`
4. Unit tests pass
5. Works with existing sample data format
